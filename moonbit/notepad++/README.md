
# Notepad++

## User Defined Language

The user defined language (UDL) allows to define rules for formatting keywords, comments, numbers, delimiters and symbols that can be used syntax highlighting and to allow folding. Here is a [customize UDL to add the MoonBit programming language](https://github.com/StSchnell/WebAssembly/blob/main/moonbit/notepad%2B%2B/userDefineLangs/moonbit.udl.xml).

## Function List

The function list panel is a zone in Notepad++ to display all the functions or methods found in the current file. The user can double-click the function name in it to jump to that function in the editor. Here is a [customized function list to add the MoonBit programming language](https://github.com/StSchnell/WebAssembly/blob/main/moonbit/notepad%2B%2B/functionList/moonbit.xml).

## Installation

Copy both files to the appropriate directories of your Notepad++ installation and restart Notepad++.<br><br>
